# flask api tut

Setup virtual env and install flask:         
```
$ cd <app>
$ virtualenv venv
$ source venv/bin/activate 
$ pip install flask
```  

To deactivate virtual env:   
```
$ deactivate
```  

To run app on http://localhost:5000 
```
$ python run.py
```
